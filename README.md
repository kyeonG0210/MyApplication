# 안드로이드 프로그래밍 수업 실습 코드

## 📚 챕터별 실습 요약

- **ch06_view**  
  뷰(View)와 뷰 바인딩을 통해 레이아웃과 코드를 연결하고, 다양한 UI 컴포넌트 활용법을 익힘

- **ch07_layout**  
  안드로이드의 다양한 레이아웃(Linear, Relative, Constraint 등) 구조와 배치 방식을 실습

- **ch08_event**  
  버튼 클릭과 같은 사용자 입력 이벤트 처리, 이벤트 리스너 및 콜백 구조를 학습

- **ch09_resource**  
  이미지, 문자열, 색상 등 리소스 파일을 관리하고 코드에서 참조하는 법을 다룸

- **ch10_dialog**  
  경고창, 날짜/시간 선택 등 기본 다이얼로그와 커스텀 다이얼로그 구현 방법을 실습

- **ch13_activity**  
  여러 액티비티 간 데이터 전달, 결과 반환, ActivityResult 등 화면 전환 구조를 배움

- **ch13_intent**  
  인텐트로 외부 앱(전화, 문자, 지도, 웹 등) 호출 및 다양한 액션과 데이터 전달 방법을 실습
