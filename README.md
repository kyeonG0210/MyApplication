## 📚 챕터별 주요 학습 내용

- **CH06 View**  
  뷰(View)와 뷰 바인딩 사용법, 코드와 레이아웃 연결 실습

- **CH08 Event**  
  버튼 클릭 등 사용자 입력 이벤트 처리와 리스너 구조 연습

- **CH09 Resource**  
  이미지·문자열 등 리소스 파일 참조/활용 방법

- **CH10 Dialog**  
  경고창, 날짜/시간 선택 등 다이얼로그 활용법 및 커스텀 팝업 생성

- **CH13 Intent & Activity**  
  여러 액티비티 간 화면 전환 및 데이터 전달  
  인텐트로 외부 앱(전화, 문자, 지도, 웹 등) 호출  
  ActivityResult, putExtra/getExtra 등 실전 예제
